Pipes Demo

Pipes are functions that run in templates and transform displayed output. They are useful for formatting, localization, slicing arrays, filtering text, and more.

Built-in pipes

  • uppercase: {{ productName | uppercase }}
  • lowercase: {{ productName | lowercase }}
  • titlecase: {{ productName | titlecase }}
  • date: {{ createdAt | date: 'dd-MMM-yyyy' }}
  • currency: {{ productPrice | currency: 'INR' : 'symbol' : '1.2-2' }}
  • percent: {{ progress | percent: '1.0-0' }}
  • json: {{ heroList | json }}
  • slice: {{ heroList | slice: 0 : 3 | json }}
  • async: {{ asyncItems | async | json }}

Custom pipes

  • truncate(20): {{ description | truncate: 20 }}
  • reverse: {{ productName | reverse }}

Custom pipe usage in component code

{{ toUpperCase(productName) }} = {{ toUpperCase(productName) }}

Real-time use case ideas

  • Sanitize and format user message previews using a truncate filter.
  • Show currency and date in locale-specific format for international users.
  • Render conditional labels with pipe chaining: {{ productName | titlecase | slice: 0 : 12 }}.
  • Auto-update UI from asynchronous data streams with async pipe.
